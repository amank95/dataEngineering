===========================================
DASHBOARD API ENDPOINTS - READY TO USE
===========================================

Base URL: http://127.0.0.1:8000

-------------------------------------------
HEALTH & STATUS
-------------------------------------------
GET http://127.0.0.1:8000/
GET http://127.0.0.1:8000/health

-------------------------------------------
LATEST & RECENT DATA
-------------------------------------------
GET http://127.0.0.1:8000/supabase/latest?limit=20
GET http://127.0.0.1:8000/supabase/recent/RELIANCE.NS?days=30

-------------------------------------------
TICKER-SPECIFIC DATA
-------------------------------------------
GET http://127.0.0.1:8000/supabase/ticker/RELIANCE.NS?start_date=2024-01-01&limit=100
GET http://127.0.0.1:8000/supabase/ticker/INFY.NS?start_date=2024-01-01

-------------------------------------------
TRAINING DATA
-------------------------------------------
GET http://127.0.0.1:8000/supabase/training-data?start_date=2024-01-01&end_date=2024-12-31
GET http://127.0.0.1:8000/supabase/training-data?start_date=2024-01-01

-------------------------------------------
ANALYTICS & INSIGHTS
-------------------------------------------
GET http://127.0.0.1:8000/supabase/top-performers?top_n=10
GET http://127.0.0.1:8000/supabase/top-performers?top_n=20&date=2024-12-31
GET http://127.0.0.1:8000/supabase/stats/RELIANCE.NS?start_date=2024-01-01
GET http://127.0.0.1:8000/supabase/stats/RELIANCE.NS?start_date=2024-01-01&end_date=2024-12-31
GET http://127.0.0.1:8000/supabase/rsi-search?min_rsi=0&max_rsi=30
GET http://127.0.0.1:8000/supabase/rsi-search?min_rsi=70&max_rsi=100

-------------------------------------------
MODEL HEALTH MONITORING
-------------------------------------------
GET http://127.0.0.1:8000/supabase/model-health?window_hours=24
GET http://127.0.0.1:8000/supabase/model-health?window_hours=48&ticker=INFY.NS
GET http://127.0.0.1:8000/supabase/drift-alerts?limit=50
GET http://127.0.0.1:8000/supabase/drift-alerts?ticker=RELIANCE.NS&feature=sma_20

-------------------------------------------
FILE DOWNLOAD
-------------------------------------------
GET http://127.0.0.1:8000/fetch-parquet

-------------------------------------------
INTERACTIVE DOCUMENTATION
-------------------------------------------
http://127.0.0.1:8000/docs
http://127.0.0.1:8000/redoc

===========================================
NOTES:
- Replace ticker symbols (RELIANCE.NS, INFY.NS) with your actual tickers
- Replace dates (2024-01-01) with your desired dates
- All endpoints return JSON (except /fetch-parquet which returns a file)
- No authentication required
